# Benzboy

A tutorial to show how to wrap an external api in to a gem.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'benzboy'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install benzboy
